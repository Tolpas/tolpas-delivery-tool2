# Tolpa's Delivery Tool

## Run locally
1. Install Node.js 18+
2. Run:
   npm install
   npm run dev

## Deploy to Vercel
1. Put this folder in a GitHub repo
2. Import that repo into Vercel
3. Deploy

## Current business logic
- Zone 1 = $25
- Rochester = $50
- Zone 2 = $100
- Outside = Ask for Quote

This is the deploy-ready prototype version.
